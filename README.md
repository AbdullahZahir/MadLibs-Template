# MadLibs-Template
A a Java application that generates interactive stories by parsing text templates containing placeholder tags and replacing them with user-provided input. The system processes predefined and custom templates, dynamically identifying missing-word markers and reconstructing complete narratives through real-time text substitution.

The application is built using an object-oriented architecture centered around a reusable MadLibs class that encapsulates template storage and processing logic. Core implementation features include string parsing, substring manipulation, index-based token detection, iterative template reconstruction, user-input handling, and dynamic placeholder resolution. The project demonstrates practical applications of text processing algorithms, object instantiation, encapsulation, and interactive command-line software design.
